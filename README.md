# Secure AI POC on Azure

This repository contains a security-first AI POC implementation designed
to pass BDF, Security, and GRC reviews.

## Highlights
- PII detection & irreversible masking
- No secrets or API keys
- Private networking only
- Full audit & cleanup automation

## Structure
- docs/: architecture & compliance
- infra/: Bicep IaC
- src/: functions & ingestion
- scripts/: deploy & cleanup
